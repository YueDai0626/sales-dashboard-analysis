# Sales Dashboard Analysis

Interactive data visualization project analyzing sales performance, trends, and business insights.

---

## Project Objective

Analyze sales data and build visual dashboards to identify revenue trends, top products, and key performance insights for business decision-making.

---

## Dashboard Insights

- Monthly revenue trend visualization
- Top-performing product categories
- Sales distribution by region
- KPI metrics overview

---

## Tools Used

- Python
- Pandas
- Matplotlib / Seaborn
- Tableau (optional dashboard version)

---

## Files

- analysis.ipynb → Data analysis & visualization notebook
- dataset.csv → Sample dataset
- dashboard.png → Visualization preview

---

## Key Skills Demonstrated

- Data cleaning
- Exploratory data analysis
- Visualization design
- Business insight extraction
